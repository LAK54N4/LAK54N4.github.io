Aplikasi Pencari Pengguna Github

Aplikasi ini berfungsi untuk mencari pengguna Github. Dibuat sebagai salah satu syarat kelulusan kelas Belajar Fundamental Aplikasi Android (BFAA) di Dicoding Indonesia. Dikerjakan pada bulan November tahun 2020 menggunakan bahasa pemrograman Kotlin dengan pola arsitektur Model View ViewModel (MVVM). Untuk mengambil data dari GitHub API menggunakan library Loopj.

Movie Catalog

Aplikasi ini berfungsi untuk melihat daftar film. Dibuat sebagai salah satu syarat kelulusan kelas Belajar Android Jetpack Pro di Dicoding Indonesia. Dikerjakan pada bulan Januari tahun 2021 menggunakan bahasa pemrograman Kotlin dengan pola arsitektur Model View ViewModel (MVVM). Data API diakses menggunakan library Retrofit.
